เปิดไฟล์ด้วย VI/Vim:
Copy code
vi filename
vim filename
เพิ่มข้อความ (Insert mode):
กด i เพื่อเข้าสู่ Insert mode และเริ่มพิมพ์
เพื่อออกจาก Insert mode ให้กด Esc
บันทึกและออกจาก VI/Vim:
กด Esc เพื่อตัวเลือก (Command mode)
พิมพ์ :wq และกด Enter เพื่อบันทึกและออก
หากต้องการออกโดยไม่บันทึก ให้พิมพ์ :q! และกด Enter
แก้ไขเนื้อหา (Command mode):
x ลบตัวอักษรที่อยู่ที่ตำแหน่งปัจจุบัน
dd ลบบรรทัดที่อยู่ที่ตำแหน่งปัจจุบัน
yy คัดลอกบรรทัดที่อยู่ที่ตำแหน่งปัจจุบัน
p วางเนื้อหาที่คัดลอกหรือตัวอักษรที่ลบ ในบรรทัดต่อจากบรรทัดปัจจุบัน
ค้นหาข้อความ (Command mode):
/keyword ค้นหาข้อความ keyword ด้านหน้าตัวเลขบรรทัด
?keyword ค้นหาข้อความ keyword ด้านหลังตัวเลขบรรทัด
กด n เพื่อค้นหาข้อความถัดไป
กด N เพื่อค้นหาข้อความที่ก่อนหน้า
ย้ายตำแหน่งเคอเซอร์ (Command mode):
h เคลื่อนไปทางซ้าย
j เคลื่อนไปทางล่าง
k เคลื่อนไปทางบน
l เคลื่อนไปทางขวา
บันทึกเปลี่ยนแปลง (Command mode):
:w บันทึกเปลี่ยนแปลงในไฟล์ (โดยไม่ออก)
:q ออกจาก VI/Vim (เมื่อไม่มีการเปลี่ยนแปลง)
:wq บันทึกและออกจาก VI/Vim
แก้ไขโดยใช้สัญญาณต่อท้าย (Vim เท่านั้น):
u เพื่อย้อนกลับการเปลี่ยนแปลงล่าสุด
Ctrl + r เพื่อทำการย้อนกลับการกลับมา
สำหรับการใช้งาน VI/Vim นั้นมี
